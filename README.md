# Tic-Tac-Toe

-This is a simple board game coded in C++.
-Two players take turn choosing a slot in a 3x3 grid to mark either "X" or "O".
-If a player marks three slots in a row (horizontally, vertically or diagnolly), they win the game.

## Features
- Two-player gameplay
- Input validation
- Win/draw detection
- Play again option

## How to Play
1. Players take turns entering a position; row and column of the desired slot. For example: 1 3
2. First player to get 3 in a row wins
3. Game ends in a draw if all spaces are filled
