# Tic-Tac-Toe

This is a simple board game coded in C++.
Two players take turn choosing a slot in a 3x3 grid to mark either "X" or "O".
If a player marks three slots in a row (horizontally, vertically or diagnolly), they win the game.

## Features
- Two-player gameplay
- Input validation
- Win/draw detection
- Play again option

## How to Play
1. Players take turns entering a position (1-9)
2. First player to occupy 3 slots in a row wins
3. Game ends in a draw if all spaces are filled with the players failing to occupy 3 slots in a row
4. Enter 'y' if players want a rematch
